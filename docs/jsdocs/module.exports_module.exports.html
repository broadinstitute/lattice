Class: exports

exports(id, verbose, offsetX, offsetY, duration)

new exports(id, verbose, offsetX, offsetY, duration)

Tooltip object constructor. This tooltip will hold a reference to an HTML element that will serve as the tooltip.
Parameters:
Name Type Default Description
id String id for the tooltip;
verbose Boolean false for debugging; set to true for console logging
offsetX Integer 10 number of pixels to offset the tooltip from the mouse cursor horizontally
offsetY Integer 12 number of pixels to offset the tooltip from the mouse cursor vertically
duration Integer 100 amount of time (in milliseconds) when transitioning tooltip states
