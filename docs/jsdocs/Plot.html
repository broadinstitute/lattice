Plot class for creating a generic 2D plot object
Parameters:
Name Type Attributes Description
data Array.<Object> array of data objects to use for charting. attributes include: - {Any} x - datapoint x-value - {Any} y - datapoint y-value - {String} c - color string - {Number} r - datapoint radius -- only used in scatterplot - {String} series - series the datapoint belongs to
type PlotType the plot type; should be one from enum PlotType
rootId String div that the SVG should be created in
userInput Object <optional>
