Class: Plot

Plot(data, type, rootId, userInput)

new Plot(data, type, rootId, userInput)

Plot class for creating a generic 2D plot object
Parameters:
Name Type Description
data Array.<Object> array of - array of data points to use for charting. attributes include: - {Any} x - datapoint x-value - {Any} y - datapoint y-value - {String} c - color string - {Number} r - datapoint radius -- only used in scatterplot - {String} series - series the datapoint belongs to
type String the plot type; should be one of the enums listed in the plots var (in constants.js)
rootId String div that the SVG should be created in
userInput Object optional, available options are described in plotConfig.js
